# **App Name**: VEO Studio Pro

## Core Features:

- Video Generation from Text: Generate videos using VEO3 based on user-provided text prompts.
- Video Generation from Still Image: Create dynamic videos by animating static images using VEO3, controlled by user prompts.
- Advanced Editing Suite: Incorporate basic video editing capabilities, to cut and merge generated and uploaded videos.
- Media Library: Implement media library functionality where users can store generated, uploaded videos, images and prompts.
- Prompt History: A simple interface for accessing the history of all used prompts. The system can suggest similar prompts using an LLM tool.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to evoke a sense of creativity and technological sophistication.
- Background color: Light Gray (#F5F5F5), providing a neutral backdrop that emphasizes the generated videos.
- Accent color: Vivid Cyan (#00BCD4), used strategically to highlight interactive elements and calls to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for headlines or body text.
- Use a set of consistent, minimalist icons throughout the interface to represent various actions and media types. The icons should be monochromatic.
- A clear and intuitive layout, optimized for user experience. Use a grid system to organize elements and ensure responsiveness across devices.
- Implement subtle animations and transitions to guide the user through the app and enhance the sense of interactivity.
